A minimalist fork of [Firefox-Mod-Blur-Modular](https://github.com/datguypiko/Firefox-Mod-Blur).

![alt text](./ASSETS/other/newdesign.png)

## Changes

- Changed address bar colors.
- Changed the default font to 'Pretendard'.
- Custom context menu styling.
- Hidden some elements from the main menu for a cleaner look.

## :wrench: Installation

> [!IMPORTANT]
>
> 1.  In `about:config`, set `toolkit.legacyUserProfileCustomizations.stylesheets` to `true`.
> 2.  Find your profile folder by going to `about:support` and clicking `Open folder` in the `Profile folder` section.
> 3.  If it doesn't exist, create a `chrome` folder inside your profile folder.
> 4.  Place `userChrome.css`, `userContent.css`, and the `ASSETS` folder into the `chrome` folder.
> 5.  Restart Firefox to see the changes.
